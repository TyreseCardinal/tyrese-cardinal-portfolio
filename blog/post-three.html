Refine Global Design System & Modular Page Structure

Date: October 24, 2025
Post #3

This update focused on refining the overall color palette, restructuring global styles, and introducing modular CSS for each page. The main goal was to make the portfolio more maintainable while ensuring a consistent visual identity across every section.

The beige and navy palette was finalized — providing a warm, professional look that fits the portfolio’s tone. Global typography, button styles, and section spacing were unified, with the layout now using global grid utilities (.grid, .grid-2, .grid-3) for responsive behavior. Page-specific stylesheets were created for the Blog, Projects, and Contact pages to keep the codebase modular.

Accessibility improvements were also introduced, including focus-visible outlines and clean visual hierarchy for text and buttons. The result is a more cohesive, readable, and scalable design system ready for future features like dark mode and the upcoming About page layout.
